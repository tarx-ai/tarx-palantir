# Sovereign Units Glossary

## Definition
A **sovereign unit** is a deployment unit (EDGE node or RACK) that runs TARX-OS under the customer’s control plane.

## Properties
- Local-first runtime + public bridge surface
- Open Enterprise/Government weights path (self-healing for private deploys)
- Closed human/local weights (user-controlled)
- Health schema + OpenAI-compatible hop
- Governance-friendly

## Apollo-class Language
We use Apollo-class lifecycle language **conceptually only**.  
There is **no live Palantir Apollo API** and **no partnership claim**.  
TARX is the complementary open runtime layer.

## What customers keep
- Full ownership of data, weights policy, and control plane
- Ability to manage the fleet their way
